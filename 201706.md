# 类数组->数组

```js
const array = [].slice(likeArray);

// better
const array = [...likeArray];
```
- [性能对比](https://soneway.github.io/js/perf.html)

# 临时数组
```js
let html = '';
for (let i = 0, len = 10; i < len; i++) {
  html += `<div data-index="${i}"></div>`;
}

// better
const html = [...new Array(10)].map((_, i) => `<div data-index="${i}"></div>`).join('');
```
